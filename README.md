# Farm Alive

## 제작 정보
- 게임 소개: 스마트 팜의 관리자가 되어 제한 시간 내에 작물을 키워 납품하는 2인용 멀티 게임
- 개발 환경
  - 개발 기간: 2024.12.12 ~ 2025.01.22
  - 개발 인원: 5명
  - 사용 툴: Unity Engine 2021.3.42f1, Visual Studio 2022, Github, XR Interaction Toolkit, Photon, Firebase
